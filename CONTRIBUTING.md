# Contributing to Town Hitbox Mod

Thanks for your interest in contributing! Here's how you can help:

## Reporting Issues

Found a bug? Please create an issue with:
- Minecraft version (1.20 or 1.21)
- Fabric version
- Steps to reproduce the bug
- Expected vs actual behavior

## Feature Requests

Have an idea? Create an issue describing:
- What feature you'd like
- Why you think it would be useful
- How you'd implement it (optional)

## Code Contributions

1. **Fork** the repository
2. **Create a branch** for your feature (`git checkout -b feature/amazing-feature`)
3. **Make your changes** and test thoroughly
4. **Commit** with clear messages (`git commit -m 'Add amazing feature'`)
5. **Push** to your fork (`git push origin feature/amazing-feature`)
6. **Open a Pull Request** with a description of your changes

## Code Style

- Use Java 17+
- Follow Minecraft/Fabric conventions
- Add comments for complex logic
- Keep methods focused and small

## Testing

Before submitting:
1. Test in Minecraft 1.20 and 1.21
2. Test with different town configurations
3. Check for performance impact
4. Verify no console errors
